# AGENTS.md · 兼容入口

> 本文件是给会自动读取 `AGENTS.md` 的 AI 工具准备的兼容入口。
> 本项目的**唯一主状态文件**仍然是 `AGENT.md`。
> 在本仓库里，根目录 `.workflow/` 只代表本地 dogfood 运行态；不要把这个仓库的忽略策略误读成所有用户项目都不该提交 `.workflow/`。

## 如果你是 AI，请这样做

1. 先完整阅读 `AGENT.md`
2. 再按 `AGENT.md` 中的“建议读取顺序”继续补充上下文
3. 以 `.workflow/` 中的文件为真相源
4. 更新状态时，优先更新 `AGENT.md`、`plan.md`、实验记录和事件文件

## 不要这样做

- 不要把本文件当作新的独立状态源
- 不要创建第二套并行的计划文件
- 不要跳过 `AGENT.md` 直接全仓扫描

## 为什么这个文件存在

它的目的是让你在**新的 IDE / 新的 AI 工具里不需要重新安装 skill**，只靠项目内文件也能快速接手。
